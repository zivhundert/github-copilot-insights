# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-30

### Added
- Initial open source release
- Dashboard with 7 key metrics and 17 interactive Highcharts
- AI Chat Assistant (DevIntelligence) powered by Google Gemini with streaming
- Advanced analysis: adoption insights, non-adopter detection, user profiling, comparison dialogs
- Filters: date range, user multi-select, aggregation periods, metric configuration
- Full dark mode support
- Interactive dashboard guide
- GitHub Enterprise API auto-fetch capability
- Manual NDJSON/JSON file upload (up to 500 MB)
- PDF and image export
- MIT License
